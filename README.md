### 模块划分

| 模块        | 释义                      |
| ----------  | ----------------------- |
| zblog-data  | 数据库映射实体类 |
| zblog-core  | 核心业务类模块，提供数据库操作、业务处理等 |
| zblog-admin | 后台博客管理模块                  |
| zblog-view  | 前台博客展示模块                    |
